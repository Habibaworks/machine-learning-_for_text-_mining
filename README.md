# Machine Learning for Text Mining 📰

Fake News Detection using NLP and Machine Learning.
Developed as a group project for the Machine Learning for Text course.

## Models Used
- TF-IDF + SVM (Baseline)
- BERT Embeddings + SVM
- DistilBERT Fine-Tuned
- Custom Attention Model (built from scratch)

## Dataset
ISOT Fake News Dataset

## Best Result
DistilBERT achieved 99.99% accuracy
